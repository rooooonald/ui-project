Getting Started

To get started, make sure you do the following steps.

Create a new HTML document and include the <meta name="viewport" /> tag. 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>DailyTube</title>
  </head>
  <body>
    <h1>This is your DailyTube page!</h1>
  </body>
</html>

As our dashboard is being built on Bootstrap CSS framework, we have to include Bootstrap JS, CSS & Bootstrap Icon Font CSS in our HTML document. Place the <link> tag in the <head> for CSS, and the <script> tag for our JavaScript bundle before the closing </body>. 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>DailyTube</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">
  </head>
  <body>
    <h1>This is your DailyTube page!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>
  </body>
</html>

On top of Bootstrap CSS, we need to include an extra CSS stylesheet and image assets in order to get the full UI experience. Simply download the zip file, extract it and put all the files in your project root.

When all files are in place, put the <link> tag in the <head> (behind Bootstrap <link> tag) for the css in your HTML document. 

<link rel="stylesheet" href="/styles/pattern.css" />

In order to utilize the chart-creating feature of D3.js, we have to also include D3 JS in the HTML document. Place the <script> tags just before the closing <head>. 

<script src="https://d3js.org/d3.v7.min.js"></script>
<script src="https://d3js.org/d3-scale-chromatic.v1.min.js"></script>

Notes

Column classes indicate the number of columns to span (e.g. .col-4 spans four). To adjust the gap size between columns, we use .p-*. 

<div class="row">
  <div class="col-4 p-2"></div>
  <div class="col-4 p-2"></div>

Breakpoints

Breakpoints are customizable widths that determine how your responsive layout behaves across device or viewport sizes.

Breakpoint Class Infix Dimensions
Extra Small None <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra Large xl ≥1200px
Double Extra Large xxl ≥1400px

Responsive Grids
