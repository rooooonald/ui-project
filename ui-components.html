Alert

An alert is a notification or warning, typically in the form of a message or signal, designed to inform someone about a specific event, condition, or situation that requires their attention or action.

I'm a SUCCESS alert.

Used for displaying success messages.

I'm an ERROR alert.

Used for displaying error messages.

I'm an INFORMATION alert.

Used for displaying information and announcements.

<div class="alert alert-success" role="alert">
  <p class="alert-heading h4 text-bold">I'm a SUCCESS alert.</p>
  <p>Used for displaying success messages.</p>
</div>

<div class="alert alert-error" role="alert">
  <p class="alert-heading h4 text-bold">I'm an ERROR alert.</p>
  <p>Used for displaying error messages.</p>
</div>

<div class="alert alert-information" role="alert">
  <p class="alert-heading h4 text-bold">I'm an INFORMATION alert.</p>
  <p>Used for displaying information and announcements.</p>
</div>

Dismissable Alerts

I'm dismissable!

Try clicking the closing button.

<div
  class="alert alert-error alert-dismissible fade show"
  role="alert"
>
  <p class="alert-heading h4 text-bold">I'm dismissable!</p>
  <p>Try clicking the closing button.</p>
  <button
    type="button"
    class="btn-close"
    data-bs-dismiss="alert"
    aria-label="Close"
  ></button>
</div>

Buttons

A button is a virtual user interface element that, when pressed or clicked, triggers an action or response in a device, application, or system. 

<button type="button" class="btn main-btn">Primary</button>
<button type="button" class="btn secondary-btn">Secondary</button>
<button type="button" class="btn alert-btn">Alert</button>

Button Sizes

If you want larger or smaller buttons, add the class .btn-*size for additional sizes.

.btn-sm
.btn-lg 
<button type="button" class="btn main-btn btn-sm">
  Small
</button>
<button type="button" class="btn main-btn btn-lg">
  Large
</button>

Modal

We use modals to display information or tips throughout the dashboard.

Example

Trigger Button

<button
  type="button"
  class="btn main-btn"
  data-bs-toggle="modal"
  data-bs-target="#ui-modal"
>
  SHOW MODAL
</button>

Modal Body

<div
  class="modal fade"
  id="ui-modal"
  tabindex="-1"
  aria-labelledby="ui-modal"
  aria-hidden="true"
>
  <div
    class="modal-dialog modal-dialog-centered modal-dialog-scrollable"
  >
    <div class="modal-content">
      <div class="modal-body">
        <h1 class="mb-3">This is an example modal.</h1>
          <p>
            Lorem ipsum dolor sit amet, consectetur adipiscing
            elit. Suspendisse aliquet diam in elit malesuada
            porttitor sed quis dolor. Aliquam mollis, metus sit
            amet lacinia congue, urna orci scelerisque lacus, vel
            tincidunt dolor tellus iaculis felis. Vestibulum
            aliquam non risus sit amet fermentum.
          </p>
        </div>
      <div class="modal-footer">
        <button
          type="button"
          class="btn secondary-btn"
          data-bs-dismiss="modal"
        >
          CLOSE
        </button>
      </div>
    </div>
  </div>
</div>
